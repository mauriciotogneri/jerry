# Jerry

## Installation

Add the following code to your **pom.xml**:

```xml
<repositories>
    <repository>
        <id>jitpack.io</id>
        <url>https://jitpack.io</url>
    </repository>
</repositories>
```

and the dependency:

```xml
<dependency>
    <groupId>com.github.mauriciotogneri</groupId>
    <artifactId>jerry</artifactId>
    <version>1.0.4</version>
</dependency>
```
